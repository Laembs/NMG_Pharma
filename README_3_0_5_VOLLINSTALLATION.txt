NMG Analyse 3.0.5 Vollinstallation

Dieses Paket enthält die aktuelle Startdatenbank inkl. gelernter Daten aus dem Projektstand.

Wichtig:
- Normale Installation überschreibt vorhandene Daten nicht.
- Für eine leere/fehlerhafte Installation kann optional installer/startdatenbank_3_0_5_einspielen.bat genutzt werden. Dabei wird vorher ein Backup erstellt.
- Der echte Windows-Installer wird unter Windows mit Inno Setup über installer/NMG_Analyse_Setup_3_0_5.iss gebaut.

Empfohlener Ablauf:
1. ZIP entpacken.
2. Inno Setup öffnen.
3. installer/NMG_Analyse_Setup_3_0_5.iss kompilieren.
4. erzeugte NMG_Analyse_Setup_3_0_5.exe installieren.
