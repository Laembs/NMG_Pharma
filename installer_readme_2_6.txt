NMG Analyse 2.6.0 - Installations- und Update-System

Wichtig:
Diese Version ist weiterhin als Prototyp-Ordner ausgeliefert. Eine echte Setup.exe kann mit build_exe.bat erzeugt werden, wenn PyInstaller installiert ist.

Update-System:
- Im Programm links auf "Update installieren" klicken.
- Eine .nmgupdate-Datei auswählen.
- Das Programm erstellt automatisch ein Backup.
- Datenbank, Backups, Ausgaben und gespeicherte Analysen werden nicht überschrieben.

Updatepaket erzeugen:
- create_update_package.bat ausführen.
- Das Paket landet im Ordner updates.

Backup:
- Beim Start wird täglich automatisch gesichert.
- Manuelle Backups bleiben erhalten.
