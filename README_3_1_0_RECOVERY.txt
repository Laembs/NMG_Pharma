# NMG Analyse 3.1.0 Recovery Edition

Diese Version ist eine saubere Neuinstallation mit wiederhergestelltem Lernstand aus Prototyp 2.6.

Enthalten:
- aktuelle Startdatenbank `nmg_startdatenbank.sqlite`
- Hersteller-/PZN-Basisdaten
- DF/PCK-Lernstand
- Austauschartikel
- historische PK-Auswertungen fuer Marktanalyse
- CSV-Hotfix
- ProgramData-Datenordner-Fix
- Setup-/Installer-Vorlage fuer Inno Setup

Wichtig:
- Programmdateien und Daten werden getrennt.
- Daten liegen unter Windows standardmaessig in `C:/ProgramData/NMG Analyse`.
- Bestehende Daten werden bei normaler Installation nicht ueberschrieben.
